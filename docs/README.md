## Overview

Welcome to the BEO (Backend Engineering Operations) Automation documentation. This comprehensive guide covers everything you need to know about setting up, using, developing, and maintaining the BEO Automation system.

## What is BEO Automation?

BEO Automation is a modern Node.js/TypeScript system designed to streamline event management and backend engineering operations workflows. It combines a powerful Next.js frontend with a robust backend, PostgreSQL database, Airtable integration, and the Oz Agent SDK to provide automated processing of business events, real-time monitoring, and seamless external integrations.

## Prerequisites

Before getting started, ensure you have:

- Node.js v18 or higher
- PostgreSQL v14 or higher
- npm or yarn package manager
- Git for version control
- Access to Airtable API (for integrations)
- Oz Agent SDK credentials (for automation workflows)
